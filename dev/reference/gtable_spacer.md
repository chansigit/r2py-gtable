# Create a row/col spacer gtable.

Create a zero-column or zero-row gtable with the given heights or widths
respectively.

## Usage

``` r
gtable_row_spacer(widths)

gtable_col_spacer(heights)
```

## Arguments

- widths:

  unit vector of widths

- heights:

  unit vector of heights

## Value

A gtable object
